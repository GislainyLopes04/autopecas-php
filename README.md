# 🔧 Sistema de Autopeças

Sistema web desenvolvido em PHP para gerenciamento de produtos de uma loja de autopeças.

## 🚀 Funcionalidades
- Login de usuário com senha segura
- Cadastro de produtos
- Listagem de produtos
- Exclusão de produtos
- Edição de produtos

## 🛠️ Tecnologias utilizadas
- PHP
- MySQL
- HTML
- CSS
- Bootstrap

## 📂 Estrutura do projeto
- config/
- controllers/
- models/
- views/

## ▶️ Como rodar o projeto
1. Instalar o XAMPP
2. Colocar a pasta do projeto em `htdocs`
3. Iniciar Apache e MySQL
4. Criar o banco de dados `autopecas_db`
5. Rodar o script SQL
6. Acessar: http://localhost/autopecas
